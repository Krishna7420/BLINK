A psychological horror survival game built with SwiftUI, SceneKit, and real-time blink detection.

In BLINK, an unknown entity approaches the player through a dark, procedurally generated environment. The only way to survive is to blink at the correct moment before the entity reaches you.

⸻

Features

* 👁️ Real-time blink detection
* 👻 Horror-inspired entity encounters
* 🌫️ Atmospheric fog and lighting effects
* ⚡ Dynamic lightning system
* 🎵 Immersive ambient audio
* 🏃 Endless survival gameplay
* 📱 Built entirely for iOS
* 🎬 Cinematic camera movement and effects
* Tech Stack


* SwiftUI
* SceneKit
* AVFoundation
* Vision Framework
* Combine

⸻

Gameplay

1. Launch the game.
2. Watch the environment carefully.
3. When an entity appears, blink before it reaches you.
4. Survive as long as possible.
5. One mistake and the entity consumes you.
